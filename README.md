# Twój Wymiar – strona stolarni

Strona internetowa firmy stolarskiej Twój Wymiar. Wersja statyczna (HTML, CSS, JS).

## Struktura

- `index.html` – strona główna
- `portfolio.html` – realizacje
- `uslugi.html` – usługi i ceny
- `o-nas.html` – o firmie
- `kontakt.html` – dane kontaktowe
- `umowienie.html` – formularz umówienia realizacji
- `styles.css` – wspólne style
- `common.js` – dark mode, nawigacja, menu mobilne

## Wdrożenie na GitHub / Vercel

```bash
cd twoj-wymiar
git init
git add .
git commit -m "Strona Twój Wymiar"
git remote add origin https://github.com/TWOJ_USER/NAZWA_REPO.git
git branch -M main
git push -u origin main
```

Po podłączeniu repozytorium do Vercel strona zostanie automatycznie wdrożona.
